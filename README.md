# litsey

Студенческое кафе «Лицей» — атмосферный React-лендинг с интерактивным меню, галереей и винтажной вечерней эстетикой.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Деплой

Сайт автоматически публикуется на GitHub Pages при пуше в `main`.
